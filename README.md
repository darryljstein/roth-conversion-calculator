# Fidelity Roth Planner

This is a simple static site for the Roth Conversion Calculator.

## Files
- `index.html` — main website file
- `vercel.json` — optional Vercel configuration

## Deploy on Vercel
1. Create a new GitHub repository.
2. Upload these files.
3. In Vercel, import the repository.
4. Click Deploy.

## Notes
- The lead form currently stores submissions in the browser only.
- To receive live email leads, connect the form to Formspree, Basin, Web3Forms, or your CRM.
- Replace the top-right placeholder with your actual logo image when ready.
